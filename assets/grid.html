آموزش عمیق و کاربردی CSS Grid

بررسی تئوری و کارگاهی مفاهیم گرید برای دانشجویان مهندسی کامپیوتر

CSS Grid چیست؟

گرید یک سیستم چیدمان (Layout) دوبعدی است که کنترل همزمان ردیف‌ها و ستون‌ها را به شما می‌دهد. برخلاف Flexbox که اساساً یک‌بعدی است، CSS Grid برای ساخت ساختارهای کلی صفحه و قالب‌های پیچیده طراحی شده است.

خطوط گرید (Grid Lines)

خطوط افقی و عمودی جداکننده که از عدد ۱ شروع شده و مرز خانه‌ها را مشخص می‌کنند.

مسیر گرید (Grid Track)

به فاصله بین دو خط موازی گرید گفته می‌شود که عملاً همان یک ردیف یا یک ستون کامل است.

سلول گرید (Grid Cell)

کوچک‌ترین واحد گرید که از تلاقی یک ردیف و یک ستون به دست می‌آید (مانند خانه‌های اکسل).

۲. ردیف‌ها و توابع کاربردی (repeat & minmax)

تعریف ردیف‌ها با grid-template-rows انجام می‌شود. برای جلوگیری از تکرار مقادیر و پیاده‌سازی مکانیزم‌های ریسپانسیو، از توابع زیر استفاده می‌شود:

تابع repeat (برای تکرار ساده): grid-template-columns: repeat(3, 1fr);
تابع minmax (محدوده امن کف و سقف ابعاد): grid-template-rows: minmax(100px, auto);
ایجاد ساختار ریسپانسیو خودکار بدون نیاز به Media Query: 
.responsive-grid {
  display: grid;
  /* ستون‌ها به عرض حداقل 200px می‌رسند 
     و مابقی فضا را پر می‌کنند */
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 16px;
}

}

۴. چیدمان بصری با grid-template-areas

قدرتمندترین متد گرید برای ساخت سریع ساختار سایت، تخصیص نام به بخش‌ها و چیدمان بصری آن‌ها در قالب کدهای متنی است:

هدر (Header)
سایدبار
محتوای اصلی سایت
فوتر (Footer)
.layout {
  display: grid;
  grid-template-columns: 200px 1fr;
  grid-template-rows: auto 1fr auto;
  /* رسم نقشه ساختار سایت */
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
}

/* انتساب کلاس‌ها به نقشه ترسیم شده */
header { grid-area: header; }
aside  { grid-area: sidebar; }
main   { grid-area: main; }
footer { grid-area: footer; }
