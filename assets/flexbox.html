آموزش عمیق و کاربردی CSS Flexbox

راهنمای جامع گام‌به‌گام برای طراحان وب و دانشجویان مهندسی کامپیوتر

Flexbox چیست؟

مدل چیدمان جعبه منعطف یا Flexbox یک سیستم چیدمان یک‌بعدی (داده‌ها به صورت یک ردیف افقی یا یک ستون عمودی قرار می‌گیرند) است. هدف اصلی این معماری، تقسیم فضای هماهنگ، ترازبندی کارآمد و توزیع شناور فضا بین آیتم‌های درون یک کانتینر است؛ حتی زمانی که اندازه آن‌ها نامشخص یا کاملاً پویا باشد.

محور اصلی (Main Axis)

محوری که فلکس‌آیتم‌ها در طول آن چیده می‌شوند. به طور پیش‌فرض افقی است مگر اینکه جهت تغییر کند.

محور فرعی (Cross Axis)

محور عمود بر محور اصلی. اگر جهت حرکت اصلی شما افقی باشد، ترازهای بالا و پایین روی این محور قرار می‌گیرند.

آیتم‌های فلکس (Flex Items)

فرزندان مستقیم یک کانتینر فلکس که بلافاصله تحت تأثیر رفتارهای کنترلی والد خود قرار می‌گیرند.

۱. کانتینر و جهت قرارگیری افقی (flex-direction: row)

حالت پیش‌فرض فلکس‌باکس که در آن آیتم‌ها در امتداد محور افقی، درست کنار یکدیگر قرار می‌گیرند:

آیتم ۱
آیتم ۲
آیتم ۳
.container {
  display: flex;
  flex-direction: row; /* رفتار پیش‌فرض افقی */
}

۲. کانتینر و جهت قرارگیری عمودی (flex-direction: column)

با تغییر محور اصلی به حالت ستونی، چیدمان به صورت عمودی (از بالا به پایین) تغییر مسیر می‌دهد:

آیتم ۱
آیتم ۲
آیتم ۳
.container {
  display: flex;
  flex-direction: column; /* چیدمان ستونی */
}

۳. ترازبندی روی محور اصلی (justify-content)

خاصیت justify-content نحوه توزیع فضاهای خالی اضافی را در طول محور اصلی کانتینر کنترل می‌کند:

تراز در مرکز (center):
A
B
بیشترین فاصله مابین آیتم‌ها (space-between):
بخش اول
بخش دوم
بخش سوم
تقسیم متقارن فضا پیرامون آیتم‌ها (space-around):
آیتم ۱
آیتم ۲
.container {
  display: flex;
  justify-content: space-between; 
  /* مقادیر دیگر: flex-start | flex-end | center | space-around | space-evenly */
}

۴. ترازبندی روی محور فرعی (align-items)

برای هم‌تراز کردن آیتم‌ها در امتداد محور فرعی (محور عمودی در حالت پیش‌فرض) از align-items استفاده می‌کنیم:

ارتفاع ۵۰px
ارتفاع ۱۲۰px
ارتفاع ۸۰px
.container {
  display: flex;
  align-items: center; 
  /* مقادیر دیگر: stretch | flex-start | flex-end | center | baseline */
}

۵. ویژگی‌های سه‌گانه منعطف (Grow | Shrink | Basis)

آیتمی که ضریب رشد بزرگ‌تری دارد، فضای خالی و آزادِ باقی‌مانده در صفحه را به صورت پویا جذب خواهد کرد:

رشد ۱ سهم
رشد ۲ سهم (دو برابر)
عرض ثابت
.item-1 { flex-grow: 1; }
.item-2 { flex-grow: 2; } /* سهم دو برابری از فضای خالی */
.item-3 { width: 80px; flex-shrink: 0; } /* جلوگیری از مچاله شدن */

۶. قابلیت انعطاف خطوط (flex-wrap)

به طور پیش‌فرض، آیتم‌ها تا حد فشرده شدن در یک خط قرار می‌گیرند. با کمک wrap به آن‌ها اجازه می‌دهیم به ردیف بعدی هدایت شوند:

آیتم الف
آیتم ب
آیتم ج
آیتم د
آیتم هـ
.container {
  display: flex;
  flex-wrap: wrap; /* پرش هوشمند به خط بعد */
}
